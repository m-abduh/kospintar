# Kospintar — Fitur

- **Multi-Property Command Center** — Punya >1 properti? Satu dashboard untuk semua properti. View per lokasi dan consolidated. Bisa dikelola dari mana aja.

- **Auto-billing + WhatsApp Reminder** — Tagihan otomatis H-7, H-3, H-1 via WA. Link pembayaran QRIS/VA. Auto-denda telat bayar.

- **AI Leasing Agent** — Otomatis balas leads 24/7 via WA, qualify calon, jadwalkan tour, kirim penawaran. Dynamic pricing: harga turun otomatis kalau okupansi rendah.

- **Financial Dashboard** — Rekap pemasukan otomatis, laporan laba-rugi per properti.

- **Ticket Management System** — Komplain penghuni otomatis jadi tiket via WA. Status real-time, auto-assign ke teknisi.

- **Auto-posting WA Status** — Foto + deskripsi kamar kosong otomatis terposting ke Status WA tiap hari.
